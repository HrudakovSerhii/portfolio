# Portfolio Website Implementation Plan

### Description:
Static website with vanilla JS, HTML and SCSS as pre-processor. Static routing, static content, translations. Modular system with utils for translation, animation, and content retreaval from files. Each translation file (file per translation) use JSON format to store key:value pairs per translation. Key use following structure: page.view.component.content, for example key=home.header.title.text and value "Portfolio", or key=main.content.portfolio.name with value "John Doe" ( portfolio owner name).
Styling will be taken from generated via figma tool values from design and will be included later.
