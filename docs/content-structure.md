# Website Content Structure

## Core Pages (4 pages maximum)
1. **Home** (`/index.html`)
   - Hero section with introduction
   - Skills overview
   - Featured projects preview
   - Call-to-action

2. **About** (`/about.html`)
   - Personal bio and background
   - Professional experience
   - Skills and technologies
   - Personal interests

3. **Projects** (`/projects.html`)
   - Project showcase with filtering
   - Project details and descriptions
   - Technologies used
   - Live demos and source links

4. **Contact** (`/contact.html`)
   - Contact form
   - Professional contact information
   - Social media links
   - Location/availability

## Navigation Structure
```
Home → About → Projects → Contact
```

## User Flow
1. **Landing**: Home page with compelling hero
2. **Discovery**: About page for personal connection
3. **Showcase**: Projects page for technical demonstration
4. **Action**: Contact page for engagement

## URL Structure (Static Routing)
- `/` - Home page
- `/about.html` - About page
- `/projects.html` - Projects page
- `/contact.html` - Contact page

## Key Content Areas Per Page

### Home Page
- Header with navigation
- Hero section (title, tagline, CTA)
- Skills preview
- Featured projects (2-3 items)
- Footer

### About Page
- Header with navigation
- Personal introduction
- Professional background
- Skills grid
- Footer

### Projects Page
- Header with navigation
- Project filter/categories
- Project grid/list
- Project modal/details
- Footer

### Contact Page
- Header with navigation
- Contact form
- Contact information
- Social links
- Footer
